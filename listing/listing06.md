Что выведет программа? Объяснить вывод программы.
```go
package main

import (
	"fmt"
)

func main() {
	var s = []string{"1", "2", "3"}
	modifySlice(s)
	fmt.Println(s)
}

func modifySlice(i []string) {
	i[0] = "3"
	i = append(i, "4")
	i[1] = "5"
	i = append(i, "6")
}
```

Ответ:
```
Вывод программы:
[3 2 3]

Слайс состоит из: ссылки на базовый массив с элементами, len(длиной) и cap(емкости)
При передачи слайса в функцию мы также передаем ссылки на элементы, на которые ссылается слайс.
При передаче его в качестве аргумента можно менять элементы текущего слайса.
Если при добавлении элемента длина увеличивается на единицу и тем самым превышает текущую вместимость,
необходимо предоставить новый объем (в этом случае текущий объем обычно удваивается). 
В этом случае будет создан новый слайс с увеличенной вместимостью и будет меняться новый слайс.

В этой программе мы передаем слайс, меняем первый его элемент на цифру 3, добавляем к слайсу 4,
его длина превышает емкость и создается новый слайс. В исходном слайсе 4 нет, в новом - есть. Его вместимость - 6.
Теперь в переменной i будет новый слайс с элементами [3 2 3 4], длиной 4 и емкостью 6
Меняем значение второго элемента нового слайса на 5 и добавляем 6. На выходе функции получается слайс: 
[3 5 3 4 6]

Но так как мы создали внтури функции новый слайс, то вне этой функции мы возвращаемся к нашему исходному слайсу, 
у которого только изменен первый его элемент
```
